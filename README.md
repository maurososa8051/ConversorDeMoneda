## Conversor de Moneda en Java

### Descripción

Este proyecto es un **conversor de monedas** desarrollado en **Java**, que permite al usuario convertir valores entre distintas divisas en tiempo real utilizando una **API de tipo de cambio** (*ExchangeRate API*).
El programa está diseñado para ejecutarse por consola y ofrece una experiencia interactiva y rápida, para practicar programación orientada a objetos.

---

###  Características

*  Conversión entre distintas monedas (USD, ARS, BRL, COP).
*  Obtención de tasas de cambio actualizadas desde una API externa.
*  Menú interactivo con ciclo repetitivo para múltiples conversiones.
*  Separación lógica del código en clases (`Conversor`, `AnalizadorJson`, `Respuesta`, `Calculadora`).
*  Control de errores al procesar datos de la API.
*  Opción para agregar historial de conversiones.

---

### Tecnologías utilizadas

*  **Java 21**
*  **Gson**
*  **ExchangeRate API**
*  **IntelliJ IDEA**
